# 1. Các điểm sai quy tắc RESTful trong API trên
    1. Sai HTTP Method:
    POST, phải là PUT/PATCH mới đúng

    2. Chứa động từ trên URL:
    /update-info là sai

    3. Sai quy tắc định danh tài nguyên
    thay vì ?id=10 thì phải là /resource/{id}
    
    4. /doctor phải là số nhiều /doctors
    
# 2. Viết lại URL này cho đúng chuẩn (về Method, Path, và Param)
    PUT /api/v1/doctors/10

# 3. Thiết kế phần Body (JSON) gửi lên để cập nhật thông tin bác sĩ sao cho các trường tuân thủ quy tắc camelCase (Ví dụ: firstName, specializationName)
